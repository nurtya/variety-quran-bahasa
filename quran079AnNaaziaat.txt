-- [Q.S. An Naazi'aat : 11]

-- [Q.S. An Naazi'aat : 12]

-- [Q.S. An Naazi'aat : 14]

-- [Q.S. An Naazi'aat : 15]

-- [Q.S. An Naazi'aat : 16]

-- [Q.S. An Naazi'aat : 18]

Dan kamu akan kupimpin ke jalan Tuhanmu agar supaya kamu takut kepada-Nya?" 
-- [Q.S. An Naazi'aat : 19]

-- [Q.S. An Naazi'aat : 20]

-- [Q.S. An Naazi'aat : 21]

-- [Q.S. An Naazi'aat : 22]

-- [Q.S. An Naazi'aat : 23]

-- [Q.S. An Naazi'aat : 24]

-- [Q.S. An Naazi'aat : 25]

-- [Q.S. An Naazi'aat : 26]

Dia meninggikan bangunannya lalu menyempurnakannya, 
-- [Q.S. An Naazi'aat : 28]

-- [Q.S. An Naazi'aat : 29]

-- [Q.S. An Naazi'aat : 30]
